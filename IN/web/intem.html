T-NGON Weight Label System

Flow vận hành: Nhân viên cân → bấm nút weight → Hub in tem ngay.
Bước 1: vào Settings cấu hình Hub URL + token + printer.
Bước 2: vào Designer thiết kế tem 50×30mm (hoặc chỉnh size).
Bước 3: vào Print nhập baseCode + giá/kg + dải gram → Generate nút → bấm để in.
Biến template
{{name}} · {{amount}} · {{weight_g}} · {{weight_kg}} · {{barcode}}
Phase 1: PNG → PDF → driver OS (ổn định, dễ triển khai)
Phase 2: in raw TSPL/ZPL (nhanh, sắc nét) — nâng cấp sau.
