# T-NGON Weight Label System (WEB)

## Deploy GitHub Pages
- Repo: `QR`
- Folder: `IN/web`
- GitHub Settings → Pages:
  - Build and deployment: Deploy from a branch
  - Branch: `main`
  - Folder: `/QR/IN/web`
- Mở link Pages trên điện thoại.

## Usage
1) Vào tab Settings:
- Hub URL: `http://192.168.58.113:8787`
- Token: `6868-XYZ-2026`
- Default Printer: chọn (hoặc để Auto)
- Test → Save

2) Vào Design & Print:
- RUN: chọn preset → bấm nút weight → in ngay
- EDIT: tạo template 50×30mm, Save template
